# CI/CD Pipeline for Attendance Application

## Overview

This project demonstrates a CI/CD pipeline for an Attendance Management Application using AWS. Source code is stored in GitHub, automatically built using AWS CodePipeline, and deployed to an Amazon EC2 instance.

## AWS Services Used

- AWS CodePipeline
- Amazon EC2
- IAM
- GitHub
- Node.js
- PM2


## Features

- Automated code deployment
- Attendance dashboard
- GitHub integration
- PM2 process management
- Continuous deployment workflow

## Deployment Steps

1. Launch Amazon EC2
2. Install Node.js and PM2
3. Create Attendance Application
4. Push Code to GitHub
5. Configure AWS CodePipeline
6. Deploy Application to EC2
7. Test Automatic Deployment
